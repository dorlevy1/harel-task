# **איך להתקין את הפרוייקט**



#### **בפרוייקט laravel לשים לב לבצע**



**composer setup**



מה שהוא עושה זה - 


php artisan key:generate


php artisan migrate


php artisan db:seed


npm install


npm run build


cd ../microservice \&\& npm install


ויוצר את קבצי הenv  מenv.example



**קובץ .env**

לשים לב למלא את הפרטים של התחברות לSQLSRV
בנוסף בתחתית העמוד נוספו

STATS\_SERVICE\_URL="http://localhost:3001"

STATS\_SERVICE\_TOKEN=super-secret-token

יש לוודא שSTATS\_SERVICE\_TOKEN תואם לmicroservice.



#### **בפרוייקט microservice לשים לב לבצע**



**קובץ .env**

לשים לב למלא את הפרטים של התחברות לSQLSRV

לשים לב שINTERNAL\_TOKEN תואם לSTATS\_SERVICE\_TOKEN בLaravel



פרטי הDB צריכים להיות מתאומים לUSER שיש לו אפשרות קריאה בלבד .



**ישנה תיקיית SQL -** 

**שבתוכה יש שאילתת SQL להכנת USER לקריאה בלבד ( שימו לב לשנות את הססמא ולהוסיף אותה בENV )**





#### **התקנה והפעלת הפרוייקט**

\---

במידה **ולא** ביצעתם את ההתקנה בהתחלה, **אז לבצע מסעיף 1.**
אם **כן** **ביצעתם** את הcomposer setup
**אפשר לגשת לסעיף 2 מראש.**



1. כניסה לתיקיית **back** ובטרמינל לכתוב **composer setup**
2. לאחר מכן להכנס חזרה לתיקיית **back** ולכתוב **composer dev** על מנת להפעיל את שלושתם.
