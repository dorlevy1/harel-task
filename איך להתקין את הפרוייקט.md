# **איך להתקין את הפרוייקט**



כניסה לתיקיית back ובטרמינל לכתוב composer setup

לאחר מכן להכנס לתיקיית microservice ולכתוב npm install

לאחר מכן להכנס חזרה לתיקיית back ולכתוב composer dev על מנת להפעיל את שלושתם.





#### **בפרוייקט laravel לשים לב לבצע** 



php artisan key:generate

php artisan migrate

php artisan db:seed



**קובץ .env** 

לשים לב למלא את הפרטים של התחברות לSQLSRV

בנוסף בתחתית העמוד נוספו

STATS\_SERVICE\_URL="http://localhost:3001"

STATS\_SERVICE\_TOKEN=super-secret-token



יש לוודא שSTATS\_SERVICE\_TOKEN תואם לmicroservice. 





#### **בפרוייקט microservice לשים לב לבצע**



**קובץ .env**

לשים לב למלא את הפרטים של התחברות לSQLSRV

לשים לב שINTERNAL\_TOKEN תואם לSTATS\_SERVICE\_TOKEN בLaravel 



פרטי הDB צריכים להיות מתאומים לUSER שיש לו אפשרות קריאה בלבד .



**ישנה תיקיית SQL - שבתוכה יש שאילתת SQL להכנת USER לקריאה בלבד ( שימו לב לשנות את הססמא ולהוסיף אותה בENV )**
